# MechJeb service

This directory contains kRPC's built-in service for interacting with
[MechJeb 2](https://github.com/MuMech/MechJeb2). MechJeb is an optional,
separately installed dependency; it is not bundled in kRPC releases.

The service was originally developed and maintained by
[Genhis](https://github.com/Genhis/KRPC.MechJeb). It was integrated into the
main kRPC repository from KRPC.MechJeb v0.8.1 at commit
`1a0575f49af55d4567c5b2c0dab60bc1d4559b00`. The archived add-on repository
retains the standalone project's earlier release history.
